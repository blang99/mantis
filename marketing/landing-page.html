How It Works

Three steps. Zero complexity.

From idea to parametric model in the time it takes to describe it.

Open Rhino, type the command, search MANTIS. One click install.

Rhino: _PackageManager
Search: MANTIS

Features

Built for how you actually design

Every feature designed for architects and computational designers.

Live Streaming Build

Components appear one-by-one as the AI generates them. Not a code dump — a live build process you can follow and learn from.

Slider: radius

Multi-Solution Explore

Generate 2-3 alternative approaches at once. Loft vs. Extrude vs. Sweep — compare and pick the best.

Error Self-Healing

Red components? MANTIS reads the errors, traces data flow, and offers AI-powered one-click fixes.

Explain Mode

Click any component chain. Get a plain-language explanation of what it does, why each piece matters, and how data flows.

100% Native Components

No GHPython black boxes. Every output is real Grasshopper components — editable, modifiable, learnable.

Multi-Model Freedom

Claude, OpenAI, Gemini, OpenRouter, Ollama. 5 providers, 15+ models. Start free, go pro when ready.

Get Started

Install in 30 seconds

No zip files. No folder copying. One command in Rhino.

01

Open Package Manager

Type in the Rhino command line:

_PackageManager
02

Search & Install

Search "MANTIS", click Install. Restart Rhino.

Search: MANTIS → Install
03

Start Building

Open Grasshopper. MANTIS auto-launches. Pick a model. Go.

Grasshopper → MANTIS panel
