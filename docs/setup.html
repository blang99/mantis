Get started

Up and building
in two minutes.

Three steps: install from Rhino's Package Manager, pick a model, and describe your first definition. Free, open source, Mac & Windows.

01

Install

In Rhino 8, run _PackageManager, search MANTIS, and click Install. Restart Rhino.

Rhino command
_PackageManager
02

Pick a model

Open the MANTIS panel. The welcome shows your model & status. Click Models to paste an API key — or choose Ollama / Claude CLI to run fully local with no key.

03

Build

Type intent in the composer — "diagrid facade on this surface" — and watch MANTIS stream its plan and place native components on your canvas.

Choose your engine

Bring a key, or stay fully local.

Bring your own key

Cloud models

Paste a key from any provider and pick the latest model. Your geometry stays local — only your prompt and chosen context are sent.

Claude · OpenAI · Gemini (free tier) · OpenRouter

No key needed

Fully local

Run an open model with Ollama, or stream from the Claude CLI with your local login. Nothing leaves your machine.

Ollama · Claude CLI

First builds

Try one of these.

"parametric tower with tapering floors"

"voronoi facade panel on this surface"

"spiral staircase, adjustable rise"

"diagrid skin, density on a slider"

"waffle structure from this surface"

"hexagonal screen with an attractor"

Requirements

Rhino 8 (Mac or Windows) with Grasshopper. MANTIS ships as a single cross-platform plugin — one install works on both. A cloud model needs an API key; local models (Ollama / Claude CLI) need neither key nor network.
